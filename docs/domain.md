# Предметная область

## Выбранная отрасль: **Розничная торговля (FMCG)**

## Клиент: Сеть супермаркетов "УниверсальМарт"
- **Масштаб**: 15 магазинов в 3 городах
- **Штат**: 450 сотрудников
- **Годовой оборот**: ~2.5 млрд руб.
- **IT-инфраструктура**: Фрагментированная, устаревшая

## Текущая ситуация
1. **Учет товаров**: Разрозненные Excel-файлы
2. **Управление запасами**: Ручные инвентаризации
3. **CRM**: Отсутствует система лояльности
4. **Аналитика**: Отчеты формируются вручную за 3-4 дня

## Цели внедрения ИС
1. Автоматизация управления товарными запасами
2. Внедрение системы лояльности клиентов
3. Централизация управления всеми магазинами
4. Сокращение операционных издержек
