#  Магазин одежды - Управленческая информационная система



##  О проекте
Система для управления магазином одежды с функциями:
- Учет товаров и остатков
- Управление продажами и закупками
- Аналитика и отчетность
- Расчет ключевых метрик (TCO, ROI, PP)

## Архитектура
- Бэкенд: Python 
- Фронтенд: Web interface
- БД: SQLite
